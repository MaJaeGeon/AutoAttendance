# AutoAttendance

<br>

코로나 바이러스로 인한 비대면 수업으로 학과게시판에 출석 인증용 댓글을 남겨야하는데  
까먹거나 잔다고 시간을 놓쳐서 출석을 못하는 경우가 생겨 이를 방지하기위해 만들었다.

<br>

## 사용 라이브러리
### Nuget Package
- [HtmlAgilityPack](https://html-agility-pack.net)  
- [Newtonsoft.Json](https://www.newtonsoft.com/json)

<br>

## 애플리케이션 정보
- .NET Core 3.1
- Console Application
